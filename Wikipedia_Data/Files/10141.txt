Title: Microsoft P-Code
Aspects: {'P-code'}

P-Code is a name for several of Microsoft's proprietary intermediate languages. They provided an alternate binary format to machine code. At various times, Microsoft have said p-code is an abbreviation for either packed code or pseudo code.Microsoft p-code was used in Microsoft Visual C++ and Microsoft Visual Basic. Like other p-code implementations, Microsoft p-code enabled a more compact executable at the expense of slower execution.
